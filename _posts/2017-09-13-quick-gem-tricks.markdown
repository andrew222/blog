## 1, 重新安装gem

```sh
gem pristine rails
```

## 2, 安装一个旧版本gem

```sh
gem install rails -v 3.2
```

## 3, Rails离线文档

``` sh
gem rdoc --rdoc --all
gem server
```

在浏览器中打开[http://localhost:8808](http://localhost:8808)
